# fortcov

A modern coverage analysis tool for Fortran code that processes coverage data generated by gfortran with `-fprofile-arcs -ftest-coverage` flags.

## Overview

fortcov generates comprehensive coverage reports in Markdown format, providing clear insights into code coverage for Fortran projects. It's designed as an MVP (Minimum Viable Product) specifically tailored for Fortran projects, similar to lcov and gcovr but with Fortran-first design principles.

## Features

- **GCC Coverage Support**: Processes `.gcda` and `.gcno` files generated by gfortran
- **Markdown Reports**: Clean, readable coverage reports in Markdown format
- **Fortran-Aware**: Understands Fortran module structures, contains blocks, and procedures
- **Line Coverage**: Accurate line-by-line coverage analysis
- **Missing Line Ranges**: Compressed display of uncovered line ranges (e.g., "10-15, 20, 25-30")
- **Modular Architecture**: Extensible design supporting multiple coverage formats and output types

## Quick Start

### Build

```bash
fpm build
```

### Generate Coverage Data

Compile your Fortran code with coverage flags:

```bash
fpm build --flag "-fprofile-arcs -ftest-coverage"
fpm test --flag "-fprofile-arcs -ftest-coverage"
```

### Generate Coverage Report

```bash
fpm run -- --format markdown --output coverage.md
```

## Architecture

fortcov follows a clean, modular architecture:

- **Coverage Model** (`coverage_model.f90`): Core data structures for coverage representation
- **Parser Abstraction** (`coverage_parser.f90`): Extensible interface for coverage format parsers  
- **Reporter Abstraction** (`coverage_reporter.f90`): Pluggable output format generators
- **File Utilities** (`file_utils.f90`): Cross-platform file operations with security focus
- **String Utilities** (`string_utils.f90`): Text processing and formatting
- **Statistics** (`coverage_statistics.f90`): Coverage calculation and analysis
- **Markdown Reporter** (`markdown_reporter.f90`): Markdown output generation
